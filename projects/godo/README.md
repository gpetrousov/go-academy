# GoDo - A Task Tracker in Golang

A project inspired by roadmap.sh https://roadmap.sh/projects/task-tracker

## Examples

```bash
# Add new tasks
go run . add Play games
go run . add Learn golang
go run . add Clean house

# List tasks
go run . list

# Mark task 1 complete
go run . mark done 1

# List done tasks
go run . list done

# Delete task 1
go run . delete 1
```
